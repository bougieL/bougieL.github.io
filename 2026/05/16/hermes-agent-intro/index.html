# Hermes Agent: 自学习 AI Agent 新范式

# 前言

在 AI Agent 领域，OpenClaw 以其强大的能力备受关注。而 Hermes Agent 作为另一个开源选择，由 Nous Research 开发，提供了独特的自学习能力。本文将带你深入了解这个"会成长的 AI Agent"。

# 什么是 Hermes Agent？

☤ 点击了解 Hermes Agent 核心特性 ▼

Hermes Agent 是 Nous Research 打造的自学习 AI Agent

  • 🧠 内置学习循环 - 从经验中创建技能并持续改进
  • 💾 记忆系统 - 跨会话持久化用户模型
  • 🔍 FTS5 搜索 - 快速检索历史对话
  • 🌐 多平台支持 - Telegram、Discord、Slack 等

# Hermes Agent vs OpenClaw

特性 Hermes Agent OpenClaw
自学习能力 ✅ 内置学习循环 ❌ 需手动配置
记忆系统 ✅ FTS5 + LLM 摘要 ⚠️ 基础会话
部署方式 $5 VPS ~ GPU 集群 云端为主
多平台支持 Telegram/Discord/Slack/WhatsApp 受限
模型支持 200+ 模型 (OpenRouter) 主要 OpenAI

# 快速安装

# Linux/macOS/WSL2/Termux
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

# Windows (PowerShell)
irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1 | iex

# 为什么选择 Hermes Agent？

  • ✅ 真正自学习 - 不只是调用工具，而是从经验中成长
  • ✅ 低成本运行 - $5 VPS 即可运行，云端空闲时几乎零成本
  • ✅ 开放标准 - 兼容 agentskills.io 开放标准
  • ✅ 研究就绪 - 支持轨迹生成和压缩，用于训练下一代模型

# 总结

Hermes Agent 代表了 AI Agent 的新方向 - 不仅能执行任务，还能从任务中学习并不断进化。如果你需要的是一个"会成长的助手"而不仅仅是工具调用器，Hermes Agent 值得关注。
