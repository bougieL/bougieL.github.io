各种视频格式的知识点

  • 创建于：2026-05-06
  • 更新于：2026-05-07

# MP4

好好想想先 😌
  • 简述
    • MP4（MPEG-4 Part 14）是最常见的视频封装格式
    • 基于 QuickTime File Format (QTFF) 发展而来
    • 支持多种编码：H.264/H.265/AV1 等
    • 兼容性最好，几乎所有设备和平台都支持

# AVI

好好想想先 😌
  • 简述
    • AVI（Audio Video Interleave）是微软推出的老牌封装格式
    • 体积大，不支持流媒体
    • 基于 RIFF 容器设计，可包含多轨道音频和视频
    • 现在已较少使用

# MKV

好好想想先 😌
  • 简述
    • MKV（Matroska）是开源的多媒体容器
    • 支持几乎所有音视频编码，灵活度高
    • 支持多音轨、多字幕、章节等高级特性
    • 常用于高清视频收藏

# MOV

好好想想先 😌
  • 简述
    • MOV 是 Apple QuickTime 的封装格式
    • 主要在 macOS/iOS 生态中使用
    • 支持高质量视频和动画
    • 基于 QTFF 框架，与 MP4 相似

# FLV

好好想想先 😌
  • 简述
    • FLV（Flash Video）是 Adobe 开发的流媒体格式
    • 曾被广泛应用于在线视频播放
    • 支持 H.264 编码，体积小
    • 随着 Flash 淘汰，逐渐被淘汰

# WebM

好好想想先 😌
  • 简述
    • WebM 是 Google 主导的免费开源格式
    • 专为 Web 设计，基于 Matroska 子集
    • 仅支持 VP8/VP9/AV1 视频和 Opus/Vorbis 音频
    • HTML5 video 标签原生支持
