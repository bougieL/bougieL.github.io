路在何方

  • 作者：Bougie
  • 创建于：2019-07-27
  • 更新于：2023-03-09
  • 成长

随着大批应届生的入职，我工作的第二年已经过去了。

# 干了些啥

戒了游戏，在咸鱼卖了以前的游戏本。现在想想以前真蠢，买那么重还不好用的笔记本干嘛。

买了心爱的 MacBook Pro，出门、开会必备。

买了一加 6T ，类安卓原生系统很喜欢，也戒了刷机病了，希望能顶个两年吧。

想想上面两个东西加起来都两万出头了，不禁也是有点心痛 😭。

终于有了一个多人开发的、开源的 GitHub 项目了，也借此能非常熟练的使用 GitHub 的各项操作了。

迁移了博客，博客内容放在数据库我不放心，放在 Git 才放心 😋。

GitHub commit 了很多次，明年看的话估计没有一千、也有八百吧。

坚持去健身房，虽说近两周太热了没去。

有了一段微妙的创业经历，虽说可能也算不上创业。

有的朋友散了就是散了，有的朋友却能结交一生。

# 变化

用一个词来形容我这一年的变化的话，我想可能用现实比较合适吧。

我时常在想，工作已经两年了。那么我和刚入行的新人的差别在哪。

以上的疑问就导致了我越来越现实，学新东西变得越来越急迫，对技术也产生了怀疑。

也许你钻研好久的技术得到结果还不如别人面试前背的一份答案。

也许你有已经精通的东西但在公司无法落地。

还也许有同事和你想法不一致，于是你放弃了自己的想法。

我感觉我的心已经很久没有静下来过了。

这是好事，它能给我动力。

# 人和人的纽带

据不完全概率统计，人一生大约会遇到 2920 万人。

做 IT 这行的，离职入职非常频繁。

也许每天和你交流业务的人，下个月就断绝了往来。

也许你觉得这个人是你的知心朋友，但是不做同一块业务后，你们就没有了任何的聊天话题。

因此，遇到真心朋友后一定要建立一个业务之外的纽带。

如果真是真心朋友的话，你一定能找到这个纽带。

# 第三年

坚持写博客。

坚持健身。

坚持每日一问。

坚持提交 GitHub。

坚持创新。

坚持自我。

...

继：前方的路
