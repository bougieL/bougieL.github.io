Vue 3.0 新特性总结

  • 作者：Bougie
  • 创建于：2019-07-01
  • 更新于：2023-03-09
  • 前端
  • Vue

# 前言

Vue 即将迎来一个大的版本升级，主要用 TypeScript 进行了重构，添加了 Function-based API。由于改动较大，目前处在 RFC(Request for Comments) 阶段。现在陆陆续续过了快一年了，还没出 3.0 正式版。

# 新特性

  1. 使用 TypeScript 进行了重构
    Vue 2.0 之前都是用的 flow 进行的类型约束，3.0 将全面采用 TypeScript

  2. 重写虚拟 Dom
    重写了虚拟 Dom 的算法，类似 React 16 的 Fiber。模板支持导出数组。

  3. 使用 Proxy 进行数据响应
